# Smart Chatbot

This WordPress plugin provides a simple chatbot capable of scraping multi-level content from a given web address and searching it for relevant information.

## Usage

1. Upload the `smart-chatbot` directory to your WordPress site's `wp-content/plugins` folder.
2. Activate **Smart Chatbot** through the Plugins menu.
3. Add the shortcode `[smart_chatbot]` to any post or page.
4. Enter a URL and a question in the chatbot interface to fetch and query content.

## Disclaimer

Scraping external websites may be subject to their terms of service. Ensure you have the right to access and process the content.
